Instalación de Angular

En esta sección te explicaremos como realizar la instalación de Angular.

  • ¿Se precisa instalación?

    • Sí, se requiere la instalación de Node.js, que incluye el gestor de paquetes npm y la interfaz de línea de comandos de Angular llamada Angular CLI.

  • ¿Qué pasos hay que dar para realizar la instalación?

    • 1. Instalación de Node.js descargando el paquete instalador desde la página oficial.

    2. Instalar Angular CLI con el comando npm install -g @angular/cli. Puede hacerse en una terminal integrada como la de VS Code.

  • ¿Versiones necesarias?

    • Lo importante aquí es que las versiones sean compatibles entre sí. La versión de Node.js actual es la 20 LTS y la de Angular es la 20.3.

  • ¿Rutas para tener en cuenta?

    • Por un lado, es importante la ruta del sistema de archivos en la que está alojado el proyecto como: C:\Users\user\Documents\tutorial_angular. Por otro lado, se deben tener en cuenta las rutas de los componentes del proyecto que permiten redirigir en la aplicación web, por ejemplo, a otras páginas o secciones.

  • ¿Variables de entorno?

    • No se necesita la declaración de variables de entorno para instalar Angular. Si Node.js y npm están instalados y se puede acceder a ellos desde la terminal, podremos instalar angular sin problema alguno.

  • ¿Necesita base de datos?

    • No se necesita la declaración de variables de entorno para instalar Angular. Si Node.js y npm están instalados y se puede acceder a ellos desde la terminal, podremos instalar angular sin problema alguno.
