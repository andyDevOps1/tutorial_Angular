Introducción a Angular

En esta sección constestaremos a varias preguntas que te ayudarán a conocer Angular.

  • ¿Para que sirve?

    • Angular es un framework de desarrollo web de código abierto mantenido por Google. Se utiliza para crear aplicaciones web dinámicas y de una sola página (SPA).

  • ¿Qué aplicaciones tiene?

    • Puede utilizarse para el desarrollo de aplicaciones web, móviles o de escritorio, sistemas de reservas, plataformas e-commerce. Permite una mantenimiento sencillo.

  • ¿Sobre qué lenguajes se apoya?

    • Es un framework de JavaScript de código abierto escrito bajo TypeScript.

  • ¿Qué prerrequisitos necesita?

    • Se requiere la instalación de Node.js, su gestor de paquetes npm o pnpm, la terminal de Angular CLI para la ejecución de ciertos comandos y un editor de código, en este caso, Visual Studio Code.
